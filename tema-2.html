Tema 2 - Grundlæggende web

I tema 2 arbejdede jeg med de grundlæggende byggesten i webudvikling, herunder HTML og CSS, designprincipper som Gestaltlovene og farveteori, Mobile First-princippet, wireframes og layoutdiagrammer. Jeg lærte at bruge både CSS Grid og Flexbox, arbejdede med responsive design via media queries og fik kendskab til værktøjer som GitHub, CodePen og style tiles.

Grundlæggende Web og Designprincipper

Vi lærte først, hvordan man indsætter indhold i HTML. Derefter fik vi en introduktion til CSS, hvor vi arbejdede med margin og padding for at styre afstand og layout. Vi fik også viden om centrale designprincipper, herunder Gestaltlovene (hvordan mennesker opfatter og grupperer visuelle elementer), farveteori (hvordan farver påvirker hinanden og skaber kontrast), og brug af webfonts til at vælge og integrere skrifttyper på en hjemmeside.

Vi fik udleveret wireframes (enkle skitser af siders struktur) og layoutdiagrammer så vi kunne se hvordan indholdet skulle placeres. Vi lavede dem altså ikke selv, men det var en god måde at se hvordan, det skulle gøres.

Studiestartsprøve og Mobile First

Vi lærte, at alle sider på et website skal have en gennemgående struktur bestående af:

  • Header - typisk med logo og navigation.
  • Main - hovedindholdet.
  • Footer - information eller links, der afslutter siden.

Efterfølgende blev vi introduceret til, hvordan man tilpasser sit design til desktopversionen ved hjælp af CSS Grid og media queries. Til sidst arbejdede vi videre på vores website, som blev til vores studiestartsprøve. Vi lærte om forskellige filformater (fx JPG, PNG, WEBP) og deres fordele. fx lærte vi om brugen af max-width i CSS for at sikre, at billeder og elementer ikke bliver for brede på store skærme. Vi fortsatte med at arbejde med CSS Grid, som jeg syntes var udfordrende i starten. Det tog mig fx lang tid at forstå forskellen på CSS grid og flexbox...

Værkstedsuge og færdiggørelse af studiestartsprøve

I værkstedsugen lavede vi lister i HTML, blev introduceret til style tiles (en visuel præsentation af farver, skrifttyper og designstil) og lærte om skrifttyper, tekstopsætning og meget mere.

Vi installerede og blev introduceret til GitHub. Vi lavede øvelser i CodePen og lærte, hvordan vi kunne bruge ChatGPT som hjælp i vores arbejde. Til sidst arbejdede vi også med at lave en burger-menu til mobilnavigation.

I forbindelse med studiestartsprøven blev vi introduceret til Mobile First-princippet, som betyder, at man starter med at designe og kode til mobilversionen først og derefter udvider layoutet til større skærme med media queries. Vi afleverede studiestartsprøven, som gik godt - jeg bestod! På billedet til højre kan man se forsiden på den hjemmeside vi skulle lave til studiestartsprøven. Hvis du holder musen over billedet, kan du klikke på det for at se hjemmesiden.

Opsummering

Alt i alt lærte jeg en masse om de mange forskellige elementer, der er med til at opbygge et website - fra den grundlæggende struktur i HTML til styling med CSS. Jeg fik viden om designprincipper og jeg lærte at bruge Mobile First-princippet og media queries til at skabe responsive løsninger, der fungerede på både mobil og desktop. Jeg arbejdede med både CSS Grid og Flexbox og lærte forskellen på, hvornår de er bedst at bruge.

Jeg fandt også ud af, hvordan jeg kunne bruge hjælpemidler som ChatGPT til min fordel, og det har virkelig været en stor hjælp - specielt til at huske de forskellige CSS-egenskaber, og hvordan de fungerer. Jeg vil ikke påstå, at jeg mestrede værktøjer som GitHub eller knækkede koden på CSS og kodning lige med det samme… Fx har jeg stadig lidt svært ved GitHub og noget kodning, men jeg synes virkelig, at jeg har lært meget, siden jeg startede.
